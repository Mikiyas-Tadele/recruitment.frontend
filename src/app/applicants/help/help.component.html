Please follow the following steps to apply for a vacancy at Development Bank of Ethiopia

  1. Please check for a vacancy in which you are eligible
  2. Register to create account to apply for a vacancy. Note: Please provide valid email address, Since your email address is used as username to log into the system
  3. After you Registered please login by providing the username and password you have created in the registration step to build your profile and apply for a vacancy.
    NB: you will only be required to build your profile once and you can apply to multiple vacancies using your profile
  4. Once you build your profile visit the active vacancies by clicking on the DBE banner to apply for the vacancy you are eligible.
  5. Clicking user profile button on the top right side displays a tab that allows you to enter your basic information, educational background, work experience, certification awarded, and finally to upload your CV
    Note that to upload a file first choose the file you want to upload and click the upload button file format should be in PDF and should not exceed 2MB
  6. The minimum data required to build your profile are, basic information, educational background, and Upload Cv
  7. To add educational background, work experience, and certification
    1. Enter the data on the form and click the save To List button so that you can enter additional multiple educational backgrounds, work experiences, and certification one by one
  8. After you have filled the minimum data required by the system click finish to save all your data
  9. After building your profile return to home and click the apply button on the list of vacancies displayed
  10. The Apply button on the right of the vacancy board takes you to a page to upload your qualification document relevant to the post applied and an area where you can type an application letter
    Note that to upload a file first choose the file you want to upload and click the upload button file format should be in PDF and should not exceed 2MB
  11. Please write an application letter on the text editor and click apply to submit your application
  12. Click the Applied Jobs button to view your application status
